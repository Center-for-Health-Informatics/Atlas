# Dependency Audit

Usage audit of every package in `package.json`'s `dependencies` (runtime deps only — `devDependencies` excluded). For each: what it's used for in Atlas, its upstream maintenance situation, and a removal/replacement verdict.

Methodology: grepped `js/` for every real import path (following `vite.config.js` aliases, since most packages are imported under an alias like `'ohdsi-api'` or `'colvis'`, not their npm name), cross-checked `package-lock.json` for transitive-only packages, and pulled `npm view <pkg> time.modified` for last-publish dates (checked 2026-07-22).

This complements `/Users/iretonrt/Projects/chi/.rob/dependency-removal.md`, which is the working log of removals already done (urijs, querystring/qs, bowser, file-saver, js-cookie, hash-it, crossfilter2) and partial native-replacement work (moment, lodash, numeral). This document is the full current-state picture; that one is the change log — check it before starting work on anything flagged below to avoid redoing analysis.

Verdict legend: **Keep** (actively needed, no better option) · **Keep, monitor** (needed but upstream is stagnant/abandoned; no action now) · **Remove** (dead weight, safe to delete) · **Candidate** (small real usage, good hand-roll target, same pattern as the moment/numeral/lodash work already underway).
